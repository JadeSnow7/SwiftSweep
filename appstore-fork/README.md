# SwiftSweep (App Store Edition)

A lightweight macOS system monitoring and disk analysis tool.

## Features

- **System Dashboard**: Real-time CPU, memory, and disk monitoring
- **Disk Analyzer**: Find large files in user-selected folders

## Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon or Intel

## Building

```bash
swift build
swift run SwiftSweepApp
```

## App Store Submission

This version is designed for App Store distribution:
- ✅ App Sandbox enabled
- ✅ Hardened Runtime
- ✅ No privileged operations
- ✅ User-consented file access only
