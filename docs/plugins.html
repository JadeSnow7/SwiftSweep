Plugin ecosystem

Safe plugins, zero code execution.

SwiftSweep plugins are data packs: secure, checksum-verified, and designed for fast updates without compromising macOS sandbox rules.

Build a Plugin Download
Data Pack Lifecycle Security-first
Discover Catalog fetch with version + compatibility metadata
Verify SHA256 checksum and optional signature validation
Activate Rules loaded into the engine, no dynamic code

Why data-only plugins?

Designed to pass Hardened Runtime and keep users safe.

Security guarantees

  • No executable code, no dynamic bundles
  • Checksum verification for every download
  • Rules run inside audited cleanup engine

Fast iteration

  • Update rules without shipping new binaries
  • Support new apps via data packs
  • Consistent UX with feature gating

Store workflow

How a plugin pack is fetched, verified, and activated.

1. Discover Fetch catalog JSON
2. Download Retrieve ZIP data pack
3. Verify Checksum + optional signature
4. Install Extract to App Support
5. Activate Load rules into engine

Manifest example

{
  "plugins": [
    {
      "id": "com.swiftsweep.capcut",
      "name": "CapCut Cleaner",
      "version": "1.0.0",
      "downloadUrl": "https://github.com/.../CapCut.rules.zip",
      "checksum": "sha256:..."
    }
  ]
}
